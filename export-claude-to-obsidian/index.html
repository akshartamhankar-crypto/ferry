How to export Claude conversations to Obsidian

Claude has no built-in export, and copy-pasting into Obsidian mangles your code blocks, tables, and artifacts. Ferry moves a whole conversation into your vault as clean Markdown in one click, and it runs entirely on your device.

The 3-step version

  1. Install Ferry (free) and set your Obsidian vault name in its settings.
  2. Open the Claude chat you want to keep and click the Ferry icon.
  3. Choose "Obsidian", Ferry hands the note straight to your vault via the obsidian:// link, frontmatter and all.
Why not just copy-paste? Pasting from Claude into Obsidian flattens fenced code (you lose the language label and indentation), turns tables into runs of text, and drops artifacts entirely. Ferry rebuilds the Markdown from the page's real structure, so a ```python block stays a ```python block.

What comes across

The full back-and-forth as ## You / ## Claude sections, fenced code with the right language, GFM tables, LaTeX math, nested lists, and YAML frontmatter (title, source, URL) so the note slots into Dataview and graph view. With Pro, every code block in the chat is also saved as a real file (.py, .js…) alongside the note.
