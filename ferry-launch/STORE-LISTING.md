## Detailed description
**Carry your AI conversations into your knowledge base — cleanly.**

Ferry exports your ChatGPT and Claude chats to Markdown that's actually usable: fenced code with the right language, tables, LaTeX, nested lists, and blockquotes all preserved — the structure that copy-paste destroys. Everything runs on your device. Ferry has no servers and never sees your conversations.

**Free**
• One-click export of any chat to clean Markdown
• Copy to clipboard, download .md, .json, or PDF
• Send straight to your Obsidian vault
• Full fidelity: code blocks, tables, math, lists

**Pro** (from $2.99 for 1 month · $14.99/year)
• One-click full-history migration — pull your entire ChatGPT or Claude history into one ZIP
• Bulk export — queue many chats into one ZIP
• Artifacts as real files — every code block / canvas saved as a correctly-typed file (.py, .html, .js…) in a folder
• Context Packs — merge chats into one Markdown bundle to re-feed into a model or project
• Notion & GitHub sync — one click, using your own keys, sent directly from your browser
• Auto-capture — archive every chat you open to an encrypted local queue

**Private by design**
No accounts. No analytics. No tracking. The only thing that ever leaves your device is the file you choose to save — and, if you turn it on, a chat you explicitly send to Obsidian, Notion, or GitHub with your own credentials. Anything stored locally is encrypted (AES-256).

Works on Chrome, Edge, Brave, Arc, and Firefox. Built by Techcesories LLC.
